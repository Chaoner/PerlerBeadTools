# 拼豆图纸工坊

一个纯前端的拼豆图片生成工具。图片在浏览器本地处理，不上传服务器。

## 使用方式

直接用浏览器打开 `index.html`。

## 已实现

- 上传 JPG / PNG / WEBP 图片。
- 支持 `image/*` 上传入口；浏览器能解码的常见图片会尽量读取，无法读取时会给出提示。
- 原图区域支持按拼豆图纸比例框选，生成时只使用框选区域。
- 支持简易版 / 详细版界面切换；简易版优先展示上传、原图和生成图，详细版保留完整编辑、材料和导出控制。
- 默认 58x58 珠位，按 29x29 拼板分区。
- 支持更多尺寸预设，包含横版、竖版和最高 500x500。
- 宽高可以独立自定义，范围为 10 到 500 颗。
- 原图始终完整展示，拼豆生成只使用完整原图上手动框选的区域。
- 支持 5mm Midi / Standard 和 2.6mm Mini。
- 色板优先提供国产常用色，并内置 Perler、Hama、Artkal 基础色板。
- 支持中文 / English 切换。
- 支持简洁图标、肖像模式、真实还原三种生成模式。
- 使用 Lab 近似色差匹配品牌色板。
- 生成时支持细节增强，会用更高精度取样、局部锐化和边缘权重尽量保留图片细节。
- 支持最大颜色数限制、透明区域空珠位、抖动开关、排除特殊材质色。
- 支持方格、圆珠、中空圆珠预览。
- 支持画笔、橡皮擦、吸管、填充、撤销、重做、恢复到生成结果、颜色替换。
- 支持库存输入、缺口统计、1000 颗单色包估算。
- 支持 PNG、CSV、PDF / 1:1 打印导出。
- 已针对手机屏幕优化单列布局、触控按钮和图纸编辑区域。

## 说明

PDF 导出使用浏览器打印流程：点击 `PDF / 1:1` 后会打开打印版页面，打印时选择“实际大小 / 100%”，即可保存为 PDF 或按 1:1 尺寸打印。
